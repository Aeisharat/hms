      This script is used to generate realistic(ish) data for use when manually testing HMS.

      It is mainly powered by data generated by http://www.fakenamegenerator.com/ Data provided by http://www.fakenamegenerator.com/ is used under the Creative Commons Attribution-Share Alike 3.0 United States license.
